# Fort School Classifier


A transparent research demonstration for testing whether the plan geometry of
a bastioned fortification resembles examples assigned to Dutch, French, or
Italian model categories. The primary demonstration is a Jupyter notebook.

The model does **not** make an architectural attribution. Historical
classification requires documentary, chronological, morphological, material,
and geographical evidence beyond an image classifier.


## Current evidence status

- The supplied `best_bastion_classifier.pth` is a valid ResNet-50 state
  dictionary with three outputs.
